Malware Analysis & Lab Environment 

This repository contains educational material presented during the ACM Cybersecurity SIG sessions, focusing on the fundamentals of malware analysis and secure infrastructure.

More specific:

Introduction to Malware & Static Analysis
An overview of core concepts for beginners in the field of reverse engineering.
- **Key Topics:** Malware classification, signature-based detection, and initial triage.
- **Techniques:** Static analysis workflows, file headers, and string obfuscation.

Isolated Lab Environment with KVM
A technical guide on building a safe, professional-grade sandbox for executing and monitoring malicious code.
- **Key Topics:** Hypervisor-level isolation, network segmentation, and snapshot management.
- **Stack:** Linux KVM (Kernel-based Virtual Machine), QEMU, and Virtual Machine Manager (virt-manager).
